# FACE RECOGNITION CHECK-IN

## 1. Dependencies
- cmake 3.22
- poetry 1.1
- python 3.8

## 2. Installation
```bash
poetry env use python3.8
poetry install
```

## 3. Training
Add face images to `images` folder

And run following command
```bash
poetry run python train.py
```

## 4. Running
```bash
poetry run python main.py
```
